<b>Cafe.ly</b> is powered by Electron + Typescript & Tailwind CSS. Its designed in a quiz-like type for users and recommends the top three coffee <i>and even non-coffee</i> drinks they can go for on their next coffee stop! Users can also look up for coffee shops that are within a certain location when searching or clicking the navigation button to automatically look for coffee shops within <i>Manila</i> instead.

* Maps is powered by OpenStreetMap, Nominatim, and Overpass, which are all open-source data for the coffee shop finder. Coffee shops are also included with various amenity tags such as <i>Free Wi-Fi, Outdoor seating, Takeaway available, etc.,</i> but not all coffee shop tags may be accurate.
* Opening hours are shown for every coffee shop, but in the event no opening hours are shown, the ❓ emoticon signifies opening hours are assumed based on usual Philippine operating hours (10:00AM-7:00PM).

# How to run

Clone the repository, and run the app locally by:
```
git clone <repository-url>
cd <repository-folder-name>
npm install
npm start
```
If `uv` is present, do not forget to `uv sync` before running the application. Otherwise, run `pip install pandas scikit-learn` for the recommendation system to work.
